Se acepta cualquiera de estos dos formatos de Excel/CSV:

Formato largo

Una fila por cada combinación jugador + partido, con estas columnas (en cualquier orden, no distinguen mayúsculas):

jornadajugadorequipo_localequipo_visitantepred_localpred_visitante

Formato ancho (por pares de equipos)

Una fila por jugador. La primera columna es el nombre, la segunda se ignora (ej. "Campeón"), y las siguientes columnas son los equipos en pares: cada dos columnas consecutivas forman un partido (equipo_local, equipo_visitante). El valor de cada celda es el marcador predicho para ese equipo; usa o para 0 goles. Indica la jornada abajo, o nombra el archivo tipo JORNADA_2.xlsx para que se detecte automáticamente.
